如何与虚拟机共享粘贴版

您需要在虚拟机关机的状态下打开virtualbox软件，在设置 -> 常规 -> 高级中，将共享粘贴版的值设为双向。

如何使用虚拟机的共享文件夹

我们已经在虚拟机中为您配置好了smb共享文件协议，虚拟机中共享文件夹的位置在\\media\share，您只需在window资源管理器的地址栏中，输入\\supernum\share即可访问该共享文件夹，访问密码与默认开机密码一致。

如果您修改了虚拟机的开机密码、主机名或共享文件夹的位置，注意替换上文中对应的信息。

在虚拟机中安装软件

为了提高工作效率，我们以最小化安装的形式为您配置了虚拟机。如果您希望虚拟机能够帮您完成更多工作，那么您可能需要打开ubuntu 软件，并从中搜索您所需要的软件。以下是我们推荐安装的应用列表：

  1. LibreOffice linux中的office套件

  2. Thunderbird 邮件收发软件

  3. DB Browser for SQLite SQLite数据库图形化管理界面

  4. MySQL Workbench MySQL数据库图形化管理软件

  5. MySQL Workbench MySQL数据库图形化管理软件

  6. DataGrip 一款全能的数据库图形化管理软件，支持多种数据库

  7. Transmission BT文件传输软件

  8. Remmina 远程连接软件

  9. shotwall 图片编辑软件

  10. Rhythmbox 音乐播放软件

  11. VLC 视频播放软件
